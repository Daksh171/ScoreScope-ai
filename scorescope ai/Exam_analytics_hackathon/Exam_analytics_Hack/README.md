# 📊 ScoreScope AI

## 🚀 Overview

ScoreScope AI is a web-based application built using Django that helps analyze student performance data. It provides insights through visualizations and analytics to improve learning outcomes.

---

## 🎯 Features

* Upload student marks (CSV / manual entry)
* Subject-wise performance analysis
* Overall score calculation
* Graphical visualization of results
* Weak subject identification
* (Optional) Machine Learning predictions

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Django (Python)
* **Database:** SQLite / MySQL
* **Libraries:** Pandas, Matplotlib

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/scorescope-ai.git
```

2. Navigate to the project:

```
cd scorescope-ai
```

3. Create virtual environment:

```
python -m venv venv
```

4. Activate environment:

* Windows:

```
venv\Scripts\activate
```

5. Install dependencies:

```
pip install -r requirements.txt
```

6. Run migrations:

```
python manage.py migrate
```

7. Start server:

```
python manage.py runserver
```
